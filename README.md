LaFab — Configurador Lutton

Landing page interactiva para la marca LaFab, enfocada en la personalización de sofás modulares tipo Lutton. Incluye un configurador en tiempo real con precios dinámicos, visualización isométrica y una experiencia editorial de alto nivel.

✨ Características
Configurador en vivo:
Número de plazas (2, 3 o 4)
Chaise longue (izquierda, derecha o ninguna)
Tipo de asiento (suave / extra suave)
Selección de textiles
Precio dinámico en tiempo real
Visualización del sofá en SVG isométrico
Diseño editorial moderno y responsive
Animaciones suaves con scroll (Intersection Observer)
UI optimizada para experiencia premium

🧱 Tecnologías utilizadas
React + TypeScript
Vite
Tailwind CSS
Hooks personalizados
SVG dinámico para render del sofá

🚀 Instalación y ejecución

1. Clonar el repositorio
git clone https://github.com/Alejot77/Prueba_LaFab.git
cd Prueba_LaFab

2. Instalar dependencias
Si usas npm:
npm install
O si usas bun (recomendado en este proyecto):
bun install

3. Ejecutar el proyecto
Con npm:
npm run dev
Con bun:
bun run dev

4. Abrir en el navegador
http://localhost:8080

⚙️ Build de producción
npm run build
o
bun run build

🧠 Notas importantes
El precio del sofá es dinámico y depende de la configuración seleccionada.
No se manejan precios fijos en el código para mantener flexibilidad.
El diseño está pensado como una experiencia premium tipo showroom digital.
